Pertanyaan Refleksi
Setelah menyelesaikan kode, jawab pertanyaan berikut:

1. Apa fungsi keyword 'extends' dalam deklarasi class Java?
2. Jelaskan perbedaan antara Method Overloading (yang ada di latihan sebelumnya) dengan Method Overriding (yang ada di latihan ini)!
3. Apa kegunaan keyword 'super' di dalam constructor subclass (Manajer)?
4. Jika variabel di class Karyawan diubah access modifier-nya menjadi 'private', apa yang akan terjadi pada class Manajer? Bagaimana solusinya?
5. Mengapa kita perlu menggunakan Inheritance? Apa keuntungan utamanya dibandingkan membuat dua class yang terpisah sama sekali (misal class Karyawan dan class Manajer dibuat terpisah tanpa hubungan)?

Jawaban:

1.Untuk mendeklarasikan bahwa sebuah class (disebut subclass atau child class) adalah turunan dari class lain (disebut superclass atau parent class).

2.Pada method overloading tidak membutuhkan inheritance, sedangkan pada overriding harus ada inheritance.

3.Untuk memanggil constructor dari superclass (kelas induk).

4.Class Manajer tidak bisa mengakses variabel tersebut secara langsung, karena private hanya mengizinkan akses dari dalam class tempat variabel yang dideklarasikan.
Solusinya:
Mengubah access modifier variabel di class Karyawan menjadi protected dan menyediakan public Getter Method.

5.Karena menyediakan struktur Is-A relationship.
Keuntungannya:
Polimorfisme dan kodenya dapat digunakan lagi.
